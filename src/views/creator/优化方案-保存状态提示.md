# 保存功能优化方案

## 当前实现
```javascript
// 创建防抖函数，延迟1秒执行
```

## 优化方案：添加保存状态提示和错误处理

### 1. 添加状态变量

在 design.vue 的 `<script setup>` 中添加：

```javascript
// 保存状态管理
const isSaving = ref(false);           // 是否正在保存
const lastSavedTime = ref(null);       // 上次保存时间
const saveError = ref(null);           // 保存错误信息
```

### 2. 优化保存函数

```javascript
// 创建防抖函数，延迟2秒执行（从1秒改为2秒，减少请求频率）
const saveToDefault = debounce(async (newValue) => {
  isSaving.value = true;
  saveError.value = null;
  
  try {
    await updateDefaultSettings(newValue);
    lastSavedTime.value = new Date();
    // 可选：显示成功提示（不打扰用户）
    // ElMessage.success({ message: '已保存', duration: 1000, showClose: false });
  } catch (error) {
    saveError.value = error.message || '保存失败';
    ElMessage.error('保存失败，请检查网络连接');
    console.error('保存失败:', error);
  } finally {
    isSaving.value = false;
  }
}, 2000);
```

### 4. 添加时间格式化函数

```javascript
// 格式化保存时间
const formatSaveTime = (time) => {
  if (!time) return '';
  
  const now = new Date();
  const diff = Math.floor((now - time) / 1000); // 秒数
  
  if (diff < 10) return '刚刚';
  if (diff < 60) return `${diff}秒前`;
  if (diff < 3600) return `${Math.floor(diff / 60)}分钟前`;
  
  // 超过1小时显示具体时间
  const hours = time.getHours().toString().padStart(2, '0');
  const minutes = time.getMinutes().toString().padStart(2, '0');
  return `${hours}:${minutes}`;
};
```

## 可选优化：手动保存按钮

如果用户想要立即保存（不等待防抖），可以添加手动保存按钮：

```vue
<template>
  <div class="save-status">
    <!-- 状态显示 -->
    <span v-if="isSaving">保存中...</span>
    <span v-else-if="lastSavedTime">已保存 {{ formatSaveTime(lastSavedTime) }}</span>
    
    <!-- 手动保存按钮 -->
    <el-button 
      size="small" 
      type="primary" 
      :loading="isSaving"
      @click="handleManualSave"
    >
      立即保存
    </el-button>
  </div>
</template>

<script setup>
// 手动保存（立即执行，不等待防抖）
const handleManualSave = async () => {
  // 取消防抖中的保存
  saveToDefault.cancel();
  
  // 立即保存
  isSaving.value = true;
  saveError.value = null;
  
  try {
    await updateDefaultSettings(questionSettings);
    lastSavedTime.value = new Date();
    ElMessage.success('保存成功');
  } catch (error) {
    saveError.value = error.message;
    ElMessage.error('保存失败');
  } finally {
    isSaving.value = false;
  }
};
</script>
```

## 可选优化：键盘快捷键保存

添加 Ctrl+S / Cmd+S 快捷键：

```javascript
// 监听键盘事件
onMounted(() => {
  const handleKeyDown = (e) => {
    // Ctrl+S 或 Cmd+S
    if ((e.ctrlKey || e.metaKey) && e.key === 's') {
      e.preventDefault(); // 阻止浏览器默认保存
      handleManualSave();
    }
  };
  
  window.addEventListener('keydown', handleKeyDown);
  
  // 清理
  onUnmounted(() => {
    window.removeEventListener('keydown', handleKeyDown);
  });
});
```
