根据我们之前讨论的，可撤销操作的清单
+ 新增题目 : 有两种新增题目的方式， 
1. **点击Design组件侧边栏的题目类型按钮**，那么将会在当前选中题目所属页面的最后新增一个响应类型的题目，API如下：
```js
addQuestionElement( 
    questionSettings,
    elementType,
    currentQuestionId.value
)
```
对应的撤销API：
```js
deleteElement(elementId: string)
```

2. **点击复制题目按钮**，会在当前点击复制按钮的题目之后，添加一个复制自身的题目，API如下：
```js
copyElement(elementId: string, elementType: string)
```
对应的撤销API：
```js
deleteElement(elementId: string)
```

+ 新增页面：
API如下
```js
addPage(
    questionSettings,
    currentQuestionId.value,
    pageIndex.value,
    isCurrentQuestionAPage.value
)
```
撤销API：
```js
handleDeletePage(questionSettings, page, index)
```

+ 拖拽页码：拖拽页码组件到新位置，有两种情况：
1. **拖拽页码到题目元素位置**，会删除原页码并在新位置插入页码，
2. **拖拽页码到页码位置**，会删除原页码并在目标页码前插入空页面


因为用vue-draggeble-next重构拖拽功能，这样或许可以规避掉手动的索引边界校验

+ 切换题目
选择类题目之间的切换题型，API如下:
```js
switchQuestionType (newType: QuestionElement["type"])
```
撤销API
```js
switchQuestionType (newType: QuestionElement["type"])
```
不过这个API的参数应该要重新考虑一下

1. 问卷的或者题目的title,description

正向操作
使用的是customEditor组件中，该组件使用TipTap组件封装，然后修改

反向操作
执行正向操作之前的title属性或者别的属性值保留，
直接用currentQuestionId找到最近正在编辑的elementId，然后通过ElementId，找到对应的element,然后重置状态

正向操作：添加选项addOption()
反向操作：删除选项deleteOption(index)
