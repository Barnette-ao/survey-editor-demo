拖拽整个题目的undo/redo
拖拽的是题目，还是页块。
如果是拖拽题目，那么是在本页内拖拽还是跨页拖拽
如果是拖拽页块，那么是在向下拖拽，是否跨页，还是向上拖拽，是否跨页/

先考虑，拖拽的是题目，而且是在本页拖拽
本页拖拽题目的可撤销，最重要的是被拖拽的题目在该页内的index的变化，
具体指，从indexA变成indexB，applyOperation此时，是从indexA -> indexB;
undo: indexB -> indexA

所以可撤销需要记录fromIndex和toIndex, 即A 和 B
