# 问卷系统启动指南

## 当前状态
✅ Mock 接口已配置
✅ API 封装已完成
✅ 问卷编辑器组件已存在

## 要跑起来还需要：

### 1. 安装 SurveyJS（必需）
```bash
npm install survey-core survey-vue3-ui
```
你的代码中使用了 `Serializer`，这是 SurveyJS 的核心功能。

### 2. 安装 Vue Router（推荐）
```bash
npm install vue-router@4
```
用于页面路由管理。

### 3. 可选依赖
- Sass/SCSS 支持（如果样式用了 scss）
```bash
npm install -D sass
```

## 快速启动方案

### 方案 A：只测试 Mock 接口（最简单）
当前 App.vue 已经配置好，直接运行：
```bash
npm run dev
```
可以看到问卷列表，测试所有 API 接口。

### 方案 B：启动问卷编辑器
需要：
1. 安装 SurveyJS
2. 配置路由
3. 创建编辑器入口页面

## 建议
先用方案 A 测试 Mock 接口是否正常工作，确认后再集成编辑器。
